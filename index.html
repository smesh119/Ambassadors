Refer friends, get rewards.

The Red Hat Ambassador program (a.k.a. Associate Referral program) rewards eligible Red Hatters who refer talented people from their networks to work at Red Hat. Learn more about the program and how to refer on the Red Hat Ambassadors Mojo page. Refer friends or apply yourself using the Associate Job Portal.

WANTED:
Senior Quality Engineer (Test Development)

The Red Hat Quality Engineering team is seeking a Senior Engineer to test our Linux platform. In this role, you'll work on a team that serves as the last gate through which code must pass on its way to public release. You'll perform function, integration, and performance testing on various tools in the Linux distribution, and track defects in system applications and daemons. We'll need you to have excellent technical skills, as you will develop test scripts and operate automated test suites to ensure the functionality of Red Hat products. This role is a great opportunity to be the first to experience new products and features before enterprise customers.
